### 🚀 Overview
An academic data mining project engineered to tackle the diagnostic challenge of time-intensive and subjective manual MRI analysis. By combining two state-of-the-art neural networks, we significantly reduced false-positive rates to meet strict medical diagnostic requirements.

### ⚡ Architecture & Performance
*   **Total Accuracy:** `98.66%`
*   **Architecture:** Dual-Branch Ensemble (`EfficientNetV2B0` + `ResNet50V2`)
*   **Dataset:** `10,464` MRI Scans (Glioma, Meningioma, Pituitary, No Tumor)
*   **Learning Strategy:** Adam Optimizer, Cosine Annealing, Cross-Entropy Loss
*   **Generalization:** 70/15/15 Data Split with Medical-Grade Augmentations
